# Command: Claim

Use claim to state what the agent believes is complete.

## Do

- Separate complete, partial, and not done.
- Attach evidence to each completion claim.
- State verification status.
- State residual risk.
- State whether memory or upgrade proposal is recommended.

## Output

```yaml
state: claim
complete:
partial:
not_done:
verification:
residual_risk:
memory_proposal_recommended:
upgrade_proposal_recommended:
```
