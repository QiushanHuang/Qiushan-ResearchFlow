# Command: Digest

Use digest to compress the execution into evidence and findings.

## Do

- List opened indexes and sources.
- List commands and important outputs.
- Separate facts, interpretations, assumptions, and unresolved questions.
- For experiments, create or reference `ResultDigest` before any `ExperimentClaim`.

## Output

```yaml
state: digest
facts:
interpretations:
assumptions:
evidence:
outputs:
candidate_claims:
candidate_memory:
```
