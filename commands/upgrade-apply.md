# Command: Upgrade Apply

Use upgrade-apply only after explicit human approval.

## Do

- Verify approved proposal.
- Apply only the approved change.
- Record files changed.
- Record rollback or supersession path.
- Re-run relevant review.

## Output

```yaml
upgrade_apply_id:
upgrade_proposal_id:
approved_by:
applied_by:
files_changed:
rollback:
review:
```
