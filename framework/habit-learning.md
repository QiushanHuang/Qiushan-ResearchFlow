# Habit Learning

User habits are learned through evidence, not imitation.

A habit observation is not memory. A habit candidate is not active behavior. Universal habit memory requires proposal, review, human approval, scope, confidence, exceptions, and invalidation conditions.

Do not infer durable habits from a single task, typo, tone, temporary constraint, or one project's local preference.

Cross-project habit reuse requires a bridge and remains scoped until approved.

## Chain

```text
HabitObservation -> HabitCandidate -> MemoryProposal -> ApprovedMemory
```

If evidence is weak, keep the record in `evolution/` as a candidate.
