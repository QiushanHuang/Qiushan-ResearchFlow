# Documentation

This directory contains public, sanitized manuals for ResearchFlow.

- [User Manual](user-manual.md): operational use from natural language prompts.
- [Technical Manual](technical-manual.md): architecture, data model, validators, and topology.
- [Local-Public Sync Manual](local-public-sync.md): release boundary and synchronization procedure.

These manuals are part of the public-sync surface. Private memory, project facts, hardware records, and local usage history belong in the local private ResearchFlow tree.

# 文档

本目录保存 ResearchFlow 的公开脱敏手册。

- [使用手册](user-manual.md)：从自然语言 prompt 开始的日常操作。
- [技术手册](technical-manual.md)：架构、数据模型、校验器和拓扑关系。
- [本地-公开同步手册](local-public-sync.md)：发布边界和同步流程。

这些手册属于可公开同步范围。私人记忆、项目事实、硬件记录和本地使用历史应保留在本地私有 ResearchFlow 树中。
